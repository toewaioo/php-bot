PHP Telegram Bot - TikTok Downloader

About This Project

This project is a PHP-based Telegram bot that allows users to download TikTok videos without a watermark. It uses the TikWM API to retrieve video URLs and the Telegram Bot API to send messages and videos to users.

Note: Ensure you have a valid Telegram Bot API token and TikWM API access to use this project.

Features

  • Download TikTok videos without watermark in HD quality.
  • Send videos directly to Telegram users.
  • Simple and efficient implementation using PHP.
  • Handles errors gracefully with user-friendly messages.

How It Works

  1. The bot listens for incoming messages on Telegram.
  2. If a message contains a TikTok URL, the bot fetches the video using the TikWM API.
  3. The video is then sent back to the user via Telegram.

Code Example

<?php
// Example function to send a message via Telegram
function sendMessage($chat_id, $text) {
  $url = API_URL . "sendMessage";
  $data = ['chat_id' => $chat_id, 'text' => $text];
  file_get_contents($url . "?" . http_build_query($data));
}
?>
